# Outfitter Philosphy

Agents are defined by their agency which is their ability to make decisions on accomplishing a task assigned to it.

# Expeditious Agents

Expeditious agents are not only effective but they are fast. Effective is measured in trust that it will acchomplish a given task as expected given a succinct prompt. Succinctiness is desirable both for human input to the agent and agentic output that the human must parse. Given the highest signal to ration going in we get a clearer (agentic) signal out, when is then amplified as sub agents are used.

# Effective Agent Decision Making

There are many timelines to a task completion, the quickiest is defined by the most direct path of decisions to task completion.

By keeping agent profiles tighly focused the agents `context window` retains a large headroom this it's decision making by keeping it's tool choises and context relevent to the decisions made in prescribed proses.

Further more this clear delination to creates a path to sub agen deligation and parallization whic feeds into our goal of expeditious agents.
